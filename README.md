# GitHub Users Viewer

A simple, fast, and responsive web application that fetches and displays GitHub users using the official GitHub API.

## 🚀 Features
- **View Users:** Displays a list of GitHub users complete with their avatars, usernames, and direct links to their profiles.
- **Search Functionality:** Filter through the loaded users dynamically simply by typing in the search bar.
- **Modern UI:** Clean, intuitive, and responsive design.

## 🛠️ Technologies Used
- **HTML5:** Building the semantic structure of the web pages.
- **CSS3:** Styling the application for a beautiful and modern look.
- **JavaScript (ES6):** Handling asynchronous fetching (`async`/`await`), DOM manipulation, and search logic.
- **GitHub API:** Source of the users' data.

## ⚙️ How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/yousefzuainat/Github-Users-Viewer.git
   ```
2. Open the project folder.
3. Open `index.html` in your web browser. 
No build steps or installations are required!

## 💡 About
This project demonstrates how to make asynchronous HTTP requests using the `fetch` API, manipulate the Document Object Model (DOM), and create a real-time search filter using Vanilla JavaScript.
